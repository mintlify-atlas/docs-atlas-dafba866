## Key Features

VertiSub CMS is a powerful WordPress-based content management system designed for enterprise organizations that need to manage complex multi-region content, educational courses, and professional services.

<CardGroup cols={2}>
  <Card title="Custom Post Types" icon="database" href="/concepts/custom-post-types">
    Manage services, courses, certifications, countries, clients, and documents with specialized content types
  </Card>
  <Card title="Multi-Region Support" icon="globe" href="/features/countries-regions">
    Handle country-specific content with interactive maps and localized contact information
  </Card>
  <Card title="Course Management" icon="graduation-cap" href="/features/courses">
    Create and manage training courses with multimedia content, instructors, and regional availability
  </Card>
  <Card title="Advanced Custom Fields" icon="sliders" href="/concepts/acf-integration">
    Flexible content management with ACF integration for custom metadata and dynamic fields
  </Card>
</CardGroup>
