This guide walks you through managing services (Servicios) in VertiSub CMS. Services represent the diving and underwater services offered by VertiSub across different countries.

## Overview

The Services post type allows you to showcase:

- Service descriptions and details
- Multimedia content (images, videos, and video links)
- Country availability
- Featured images

## Creating a New Service

<Steps>

### Navigate to Services

In the WordPress admin dashboard, click **Servicios** in the left sidebar (look for the hammer icon).

### Add New Service

Click **"Añadir servicio"** at the top of the page.

### Enter Service Title

Add the service name in the title field (e.g., "Underwater Inspections", "Commercial Diving", "Marine Construction").

### Add Service Description

Use the main content editor to write a comprehensive description of the service. This supports:

- Text formatting (bold, italic, headings)
- Lists and bullet points
- Links and embedded content
- WordPress blocks

### Set Featured Image

In the right sidebar, find the **"Imagen destacada"** panel:

1. Click "Establecer la imagen destacada"
2. Upload a new image or select from the Media Library
3. Choose a high-quality image that represents the service
4. Click "Establecer la imagen destacada" to confirm

</Steps>

## Adding Multimedia Content

The **Multimedia** meta box allows you to add multiple images, videos, and video links to showcase your service.

### Adding Images

<Steps>

### Locate the Multimedia Box

Scroll down below the main editor to find the **"Multimedia"** meta box.

### Add Image Field

Under the **"📷 Imágenes"** section, click the **"+ Agregar Imagen"** button.

### Upload Image

Click the **"Subir"** button next to the new text field to open the WordPress Media Library.

### Select or Upload

- Upload a new image by dragging and dropping or clicking "Subir archivos"
- Or select an existing image from the Media Library
- Click **"Usar esta imagen"** to confirm

### Add More Images

Repeat for each additional image. The image URL will automatically populate in the text field.

### Remove Images

Click the **"Eliminar"** button next to any image field to remove it.

</Steps>

### Adding Video Files

<Steps>

### Add Video Field

Under the **"🎥 Videos (archivos)"** section, click the **"+ Agregar Video"** button.

### Upload Video

Click the **"Subir"** button to open the media selector.

### Select Video File

Upload a video file from your computer or select from the Media Library. Supported formats include MP4, MOV, AVI.

### Confirm Selection

Click **"Usar este video"** to add the video URL to the field.

</Steps>

<Note>
Video files can be large. Consider hosting videos on YouTube or Vimeo and using video links instead for better performance.
</Note>

### Adding YouTube/Vimeo Links

<Steps>

### Add Video Link Field

Under the **"🌐 Links de Video (YouTube/Vimeo)"** section, click the **"+ Agregar Link"** button.

### Enter Video URL

Paste the full URL of your YouTube or Vimeo video into the text field.

Examples:
- `https://www.youtube.com/watch?v=VIDEO_ID`
- `https://vimeo.com/VIDEO_ID`

### Add More Links

Click **"+ Agregar Link"** for each additional video.

### Remove Links

Click **"Eliminar"** next to any link to remove it.

</Steps>

## Assigning Countries

Connect your service to specific countries where it's available.

<Steps>

### Find Countries Panel

In the right sidebar, locate the **"Países"** meta box.

### Select Countries

The box displays a multi-select dropdown showing all available countries (from the Países post type).

### Choose Multiple Countries

- **Windows/Linux**: Hold Ctrl and click to select multiple countries
- **Mac**: Hold Cmd (⌘) and click to select multiple countries
- Or click and drag to select a range

### Verify Selection

Selected countries will be highlighted in the list.

</Steps>

<Warning>
You must first create country entries in the **Países** section before they appear in this list. See the [Managing Locations](/guides/managing-locations) guide.
</Warning>

## Publishing Your Service

<Steps>

### Review Content

Scroll through all sections to ensure you've completed:

- ✓ Service title
- ✓ Service description
- ✓ Featured image
- ✓ Multimedia content (if applicable)
- ✓ Country assignments

### Preview (Optional)

Click the **"Vista previa"** button to see how your service will appear on the live site.

### Publish

Click the **"Publicar"** button in the right sidebar to make your service live.

</Steps>

## Editing an Existing Service

<Steps>

### Navigate to Services List

Click **Servicios** in the left sidebar.

### Find Your Service

Browse the list or use the search box to find the service you want to edit.

### Open Editor

Hover over the service title and click **"Editar"**.

### Make Changes

Update any fields as needed.

### Update

Click **"Actualizar"** to save your changes.

</Steps>

## Service Management Tips

### Image Recommendations

- Use landscape-oriented images (16:9 ratio works best)
- Minimum resolution: 1200 x 675 pixels
- Keep file sizes under 500KB for optimal loading speed
- Use descriptive file names before uploading

### Video Best Practices

- Prefer YouTube/Vimeo links over uploaded video files for better performance
- Ensure videos are set to "Public" or "Unlisted" (not Private) on the hosting platform
- Add multiple videos to create a service gallery
- Keep videos under 3 minutes for better engagement

### Country Association

- Only select countries where the service is actively available
- Update country associations when expanding to new locations
- Use the same country entries across services and courses for consistency

### Content Organization

- Use clear, customer-focused service titles
- Write descriptions that answer: What is it? Who is it for? What are the benefits?
- Update services regularly to reflect current offerings
- Archive or delete discontinued services

## Common Issues

### Multimedia Not Saving

If multimedia content isn't being saved:

1. Check that you clicked the "Subir" button to actually upload files
2. Verify file formats are supported (JPG, PNG, GIF for images; MP4, MOV for videos)
3. Ensure file sizes don't exceed your server's upload limits
4. Save the service as a draft first, then add multimedia

### Countries Not Appearing

If the countries list is empty:

1. Navigate to **Países** in the sidebar
2. Create at least one country entry
3. Return to your service and refresh the page
4. The countries should now appear in the dropdown

### Preview Not Showing Content

If preview shows incomplete content:

1. Save your changes first (click "Guardar borrador")
2. Then click preview
3. Ensure your theme templates are properly configured
