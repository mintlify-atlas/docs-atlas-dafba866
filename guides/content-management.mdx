This guide covers the fundamental content management workflows in the VertiSub CMS WordPress theme, helping you efficiently create and organize content across different post types.

## Understanding Content Types

VertiSub CMS provides several specialized content types (Custom Post Types) to organize your content:

- **Servicios (Services)**: Diving and underwater services offered by VertiSub
- **Cursos (Courses)**: Training programs and certifications
- **Países (Countries)**: Location information and contact details for each country
- **Documentos por País**: Country-specific documents and resources
- **Clientes (Clients)**: Client testimonials and case studies
- **Certificaciones**: Certifications and accreditations

Each content type has its own dedicated section in the WordPress admin dashboard, accessible from the left sidebar menu.

## General Content Management Workflow

<Steps>

### Navigate to Content Type

In the WordPress admin dashboard, locate the content type you want to manage in the left sidebar menu. Each content type has a distinctive icon:

- Servicios: Hammer icon
- Cursos: Awards icon
- Países: Location icon
- Documentos por País: Document icon

### Create New Content

Click on the content type name, then click "Add New" at the top of the page or in the submenu.

### Fill in Basic Information

Every content type requires basic information:

- **Title**: The name of your service, course, country, etc.
- **Content Editor**: The main description or content area (uses WordPress block editor)
- **Featured Image**: A primary image that represents this content (when supported)

### Configure Custom Fields

Below the main editor, you'll find custom meta boxes specific to each content type. These contain additional fields that enhance your content with structured data.

### Save or Publish

Click "Publish" to make the content live, or "Save Draft" to continue working on it later.

</Steps>

## Working with Featured Images

Many content types support featured images (also called thumbnails):

<Steps>

### Locate Featured Image Panel

In the right sidebar, find the "Featured Image" panel (you may need to scroll down).

### Set Featured Image

Click "Set featured image" to open the WordPress Media Library.

### Upload or Select

- **Upload New**: Drag and drop an image or click "Upload files"
- **Select Existing**: Choose from previously uploaded images in the Media Library

### Configure Image

Optionally add alt text and caption for better accessibility and SEO.

### Confirm Selection

Click "Set featured image" to attach the image to your content.

</Steps>

## Content Organization Best Practices

### Consistent Naming

Use clear, descriptive titles that help both users and search engines understand your content.

### Complete All Fields

Fill in all available fields to provide comprehensive information. Empty fields may result in missing information on the frontend.

### Use Quality Images

- Upload high-resolution images (minimum 1200px wide recommended)
- Use descriptive file names before uploading
- Always add alt text for accessibility

### Review Before Publishing

Use the "Preview" button to see how your content will appear on the live site before publishing.

## Editing Existing Content

<Steps>

### Access Content List

Click on the content type name in the sidebar to view all existing items.

### Find Your Content

Use the search box or browse the list to find the content you want to edit.

### Edit Content

Hover over the title and click "Edit" to open the editor.

### Make Changes

Update any fields as needed.

### Update

Click "Update" to save your changes.

</Steps>

## Managing Content Status

WordPress provides several status options for your content:

- **Published**: Live and visible on your website
- **Draft**: Saved but not published
- **Pending Review**: Awaiting approval from an editor or administrator
- **Scheduled**: Set to publish automatically at a future date/time

## Bulk Actions

When managing multiple items:

<Steps>

### Select Multiple Items

Check the boxes next to the content items you want to modify.

### Choose Bulk Action

Select an action from the "Bulk Actions" dropdown (Move to Trash, Edit, etc.).

### Apply

Click "Apply" to execute the action on all selected items.

</Steps>

## Tips for Efficient Content Management

### Use Screen Options

Click "Screen Options" at the top of any content list to customize which columns are displayed and how many items appear per page.

### Quick Edit

Hover over any content item and click "Quick Edit" to modify basic fields without opening the full editor.

### Duplicate Content

When creating similar content, consider using a duplication plugin to copy existing content and modify it, saving time on repetitive data entry.

### Regular Backups

Ensure your WordPress site has regular backups configured before making bulk changes or deletions.
